# CourseComrade
### Installation
Install project dependecies:
```bash
$ npm install
```
---
### Launch App
Start the backend and frontend servers:
```bash
$ npm start
```
---
### Testing Endpoints 
Test controller and api methods
```bash
$ npm test
```
---
